About This Tool

A standalone rent roll underwriting tool for office investment sales analysts. Upload any rent roll and get instant analysis across five tabs.

📊 Dashboard — KPIs, flags, charts
📋 Rent Roll — editable underwriting grid
🔄 Rollover & Costs — TI/LC by year
📄 UW Summary — printable one-pager

Column auto-detection
The tool scans for common column names. It handles files with title rows above the header (e.g. property name on row 1, column headers on row 2).

Step 2 — Rent Roll TabEdit assumptions
Editable columns
Mkt $/SF — market rent per tenant. Drives MTM calculation and flags.
TI $/SF — tenant improvement allowance. Use Auto-Generate to seed values, then adjust.
LC $/SF — leasing commission per SF.
Credit Rating — type S&P rating (e.g. A-, BBB+). Self-colors green/amber/red.
Lease Type — NNN / Modified Gross / Gross / Full Service.
